## Chapter 6: The AI Revolution: Generative AI and Azure OpenAI — Flashcards

| # | Front (Question) | Back (Answer) |
|---|-----------------|---------------|
| 1 | What is Generative AI? | AI that creates new, original content (text, images, code, audio) by learning patterns from existing data rather than just analyzing or classifying information |
| 2 | What are Large Language Models (LLMs)? | Deep learning models trained on massive text datasets that can understand and generate human-like text, forming the foundation of modern generative AI |
| 3 | What does GPT stand for and what is it? | Generative Pre-trained Transformer - a type of LLM developed by OpenAI that generates human-like text based on input prompts |
| 4 | What is the Transformer architecture? | A neural network architecture using self-attention mechanisms to process sequential data in parallel, enabling models to understand context and relationships in text |
| 5 | What is prompt engineering? | The practice of designing and refining input prompts to get optimal, accurate, and relevant responses from generative AI models |
| 6 | What is Azure OpenAI Service? | A Microsoft Azure service providing enterprise-grade access to OpenAI's powerful models (GPT, DALL-E, Codex) with Azure's security, compliance, and networking capabilities |

| 8 | What are Copilot experiences in Microsoft products? | AI-powered assistants embedded in Microsoft tools (Word, Excel, GitHub) that help users create content, write code, and complete tasks using natural language |
| 9 | What are three key principles of responsible generative AI use? | Fairness (avoiding bias), transparency (disclosing AI use), safety (preventing harmful content), and accountability (human oversight) |
| 10 | What are multimodal capabilities in AI? | The ability of AI models to process and generate multiple types of data (text, images, audio, video) within a single model |
| 11 | What is a "system message" in prompt engineering? | Instructions given to the AI that define its behavior, personality, and constraints for the conversation |
| 12 | Which model in Azure OpenAI generates images from text descriptions? | DALL-E |
| 13 | What is "grounding" in the context of generative AI? | Providing the AI with specific context or data to base its responses on, reducing hallucinations and improving accuracy |
| 14 | What is a "hallucination" in generative AI? | When an AI model generates content that sounds plausible but is factually incorrect or completely fabricated |
| 15 | Why choose Azure OpenAI Service over direct OpenAI API? | Enterprise security, compliance certifications, private networking, regional availability, integration with Azure services, and responsible AI content filtering |

### Key Terms

| Term | Definition |
|------|-----------|
| Large Language Model (LLM) | A deep learning model trained on vast text data to understand and generate human language |
| Transformer | Neural network architecture using attention mechanisms for parallel processing of sequential data |
| Prompt Engineering | Crafting effective inputs to optimize AI model outputs |
| Token | The basic unit of text that LLMs process (roughly 4 characters or 3/4 of a word in English) |
| Temperature | A parameter controlling randomness in AI outputs; lower = more focused, higher = more creative |
| Multimodal | AI capable of processing and generating multiple data types (text, image, audio) |
| Hallucination | AI-generated content that is plausible-sounding but factually incorrect |

### Memory Tricks
- **GPT = "Generate Pretty Text"** - Remember GPT creates text content from prompts
- **DALL-E = "Draw ALL Everything"** - DALL-E is for image generation (drawing)
- **Think of prompts like a recipe** - Better ingredients (clearer prompts) = better dish (output). System message is the cooking style, user prompt is the order
- **Temperature like a stove** - Low heat (0) = precise/consistent results; High heat (1) = wild/creative results
- **Transformer = "Attention Please!"** - The key innovation is the attention mechanism that helps the model focus on relevant parts of input
