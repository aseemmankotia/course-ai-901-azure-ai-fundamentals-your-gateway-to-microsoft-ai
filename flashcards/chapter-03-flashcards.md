## Chapter 3: Teaching Machines to Learn: ML Fundamentals — Flashcards

| # | Front (Question) | Back (Answer) |
|---|-----------------|---------------|
| 1 | What are features in machine learning? | Input variables or attributes used to make predictions (e.g., square footage, number of bedrooms for house price prediction) |
| 2 | What are labels in machine learning? | The output values or answers the model is trying to predict (e.g., house price, spam/not spam) |
| 3 | What is the purpose of a training dataset? | To teach the model by allowing it to learn patterns from labeled examples |
| 4 | What is a validation dataset used for? | To evaluate model performance during training and tune hyperparameters without touching test data |
| 5 | What is supervised learning? | ML technique where models learn from labeled data (inputs paired with correct outputs) |
| 6 | When would you use regression? | When predicting continuous numerical values (e.g., temperature, price, age) |
| 7 | When would you use classification? | When predicting discrete categories or classes (e.g., spam/not spam, cat/dog/bird) |
| 8 | What is unsupervised learning? | ML technique where models find patterns in unlabeled data without predefined answers |
| 9 | What is clustering used for? | Grouping similar items together based on shared characteristics without predefined labels |
| 10 | What makes deep learning "deep"? | Multiple layers of artificial neurons (hidden layers) in neural networks |
| 11 | What is the transformer architecture primarily known for? | Processing sequential data using attention mechanisms, foundational for modern LLMs like GPT |
| 12 | What does accuracy measure in model evaluation? | The percentage of correct predictions out of total predictions |
| 13 | What is Azure Machine Learning? | Microsoft's cloud platform for building, training, deploying, and managing ML models |
| 14 | How do you choose between the three main ML techniques? | Regression for numbers, Classification for categories, Clustering when you have no labels |
| 15 | What is the attention mechanism in transformers? | A technique that allows the model to focus on relevant parts of input when making predictions |

### Key Terms

| Term | Definition |
|------|-----------|
| Features | Input variables/attributes used by ML models to make predictions |
| Labels | Target output values the model learns to predict |
| Regression | Supervised learning technique that predicts continuous numerical values |
| Classification | Supervised learning technique that predicts discrete categories |
| Clustering | Unsupervised learning technique that groups similar items together |
| Neural Network | Computing system inspired by biological brains with interconnected nodes/neurons |
| Transformer | Architecture using attention mechanisms for processing sequential data |
| Validation Dataset | Data subset used to evaluate and tune model during training |

### Memory Tricks
- **RCC Rule**: "**R**egression = **R**eal numbers, **C**lassification = **C**ategories, **C**lustering = **C**lueless (no labels)"
- **Features vs Labels**: Think of a recipe — Features are the **ingredients** (inputs), Labels are the **finished dish** (output)
- **Supervised vs Unsupervised**: Supervised has a **teacher** (labels), Unsupervised is **self-study** (finds patterns alone)
- **TRAIN-VALID-TEST**: Think "**T**each, **V**erify, **T**est" — like studying, practice quiz, then final exam
