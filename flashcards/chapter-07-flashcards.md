## Chapter 7: Meet Microsoft Foundry: Your AI Development Hub — Flashcards

| # | Front (Question) | Back (Answer) |
|---|-----------------|---------------|
| 1 | What is Microsoft Foundry? | A unified platform (formerly Azure AI Studio) for building AI solutions that provides access to pre-built models, deployment tools, and SDKs to create end-to-end AI applications |
| 2 | What was Microsoft Foundry previously called? | Azure AI Studio |
| 3 | What is the Model Catalog in Microsoft Foundry? | A collection of pre-built AI models from Microsoft and partners that you can browse, evaluate, and deploy for your applications |
| 4 | What are the main components you can access through the Foundry portal? | Model catalog, project management, deployment tools, evaluation metrics, and prompt flow capabilities |
| 5 | What is a Project in Microsoft Foundry? | A container that organizes your AI resources, deployments, data, and configurations for a specific AI application or solution |
| 6 | What is the Foundry SDK used for? | To programmatically interact with Microsoft Foundry services, deploy models, and integrate AI capabilities into applications using code |
| 7 | What is Azure Content Understanding? | A service that helps extract insights and information from various content types including documents, images, and other unstructured data |
| 8 | How do you deploy a model in Microsoft Foundry? | Select a model from the catalog, configure deployment settings (compute, scaling), and deploy to an endpoint for application access |
| 9 | What types of models can you find in the Foundry Model Catalog? | OpenAI models, Meta Llama models, Mistral models, Microsoft models, and various open-source models |
| 10 | What is a deployment endpoint in Foundry? | A URL that provides access to your deployed model, allowing applications to send requests and receive AI-generated responses |
| 11 | What is a lightweight AI application in the Foundry context? | A simple AI-powered application that uses pre-built models without requiring extensive custom training or infrastructure management |
| 12 | What programming languages does the Foundry SDK support? | Python and JavaScript/TypeScript are the primary supported languages |
| 13 | What is the benefit of using the Foundry portal for navigation? | It provides a visual interface to explore models, manage projects, monitor deployments, and configure AI solutions without writing code |
| 14 | How does project management work in Microsoft Foundry? | Projects contain all resources for an AI solution including connections, deployments, data assets, and evaluation results in one organized workspace |
| 15 | What is the key takeaway of Microsoft Foundry? | It's your unified platform providing access to pre-built models, deployment tools, and SDKs to create complete end-to-end AI applications |

### Key Terms

| Term | Definition |
|------|-----------|
| Microsoft Foundry | Unified AI development platform for building, deploying, and managing AI solutions |
| Model Catalog | Repository of pre-built AI models available for deployment and customization |
| Foundry SDK | Software development kit for programmatically accessing Foundry services |
| Project | Organizational container for AI resources, deployments, and configurations |
| Deployment Endpoint | URL providing API access to a deployed AI model |
| Azure Content Understanding | Service for extracting insights from unstructured content like documents and images |
| Foundry Portal | Web-based interface for navigating and managing AI development resources |

### Memory Tricks
- **FOUNDRY = Find, Organize, Understand, Navigate, Deploy, Run, Yield** - the steps of building AI apps in the platform
- **Think of a real foundry** - just as a metal foundry shapes raw materials into useful products, Microsoft Foundry shapes AI models into useful applications
- **Model Catalog = AI App Store** - browse and "download" (deploy) pre-built AI models like shopping for apps
- **Project = Toolbox** - keeps all your AI tools and resources organized in one place for each job
