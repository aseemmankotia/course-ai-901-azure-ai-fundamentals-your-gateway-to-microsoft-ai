## Chapter 9: Exam Day Success: AI-901 Review and Test Strategies — Quick Reference

### Core Concepts
| Concept | One-line explanation |
|---------|---------------------|
| AI-901 Exam Format | 40-60 questions, 45 minutes, passing score ~700/1000 |
| Domain Weighting | AI concepts (40-45%) + Foundry implementation skills (55-60%) |
| Multiple Choice | Single/multiple answer questions testing concept knowledge |
| Case Studies | Scenario-based questions requiring applied problem-solving |
| Drag-and-Drop | Match or order items to test workflow understanding |

### Key Syntax / Commands
```
FIVE EXAM DOMAINS:
1. Describe AI workloads and considerations (15-20%)
2. Describe fundamental principles of ML on Azure (20-25%)
3. Describe features of Computer Vision workloads (15-20%)
4. Describe features of NLP workloads (15-20%)
5. Describe features of generative AI workloads (15-20%)
```

### Common Patterns
**Pattern 1: Process of Elimination**
Read all options first, eliminate obviously wrong answers, then compare remaining choices

**Pattern 2: Keyword Spotting**
Look for qualifiers like "always," "never," "best," "first" — these often signal trap answers

### Things to Remember
✅ Focus on WHY services exist, not just WHAT they do
✅ Budget ~45 seconds per question, flag difficult ones and return later
✅ Review all five domains equally — no section is optional
✅ Get hands-on practice in Azure AI Foundry before exam day
❌ Don't memorize facts without understanding practical applications
