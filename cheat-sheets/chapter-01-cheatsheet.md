## Chapter 1: Welcome to the World of AI: What Makes Machines Smart? — Quick Reference

### Core Concepts
| Concept | One-line explanation |
|---------|---------------------|
| Artificial Intelligence (AI) | Software that mimics human cognitive abilities like learning, reasoning, and problem-solving |
| Machine Learning (ML) | Subset of AI where systems learn patterns from data without explicit programming |
| Deep Learning (DL) | Subset of ML using neural networks with multiple layers for complex pattern recognition |
| Prediction | AI workload that forecasts outcomes based on historical data patterns |
| Anomaly Detection | Identifies unusual patterns or outliers that deviate from expected behavior |
| Computer Vision | Enables machines to interpret and analyze images and video content |
| Natural Language Processing (NLP) | Allows machines to understand, interpret, and generate human language |
| Generative AI | Creates new content (text, images, code) based on learned patterns |

### Key Syntax / Commands
```
AI Hierarchy:  AI → Machine Learning → Deep Learning

Workload Selection Formula:
- Structured data + future outcomes = Prediction
- Structured data + unusual patterns = Anomaly Detection
- Images/video = Computer Vision
- Text/speech = NLP
- Create new content = Generative AI
```

### Common Patterns
**Pattern 1: Workload Matching**
Identify the input type (text, image, numbers) and desired output to select the appropriate AI workload.

**Pattern 2: Azure AI Services Selection**
Match business need → AI workload type → Specific Azure service (Vision, Language, Speech, OpenAI, etc.)

### Things to Remember
✅ AI is the umbrella term; ML and DL are progressively specialized subsets
✅ Always start by identifying the business problem before selecting an AI workload
✅ Azure AI services are pre-built, making AI accessible without deep expertise
❌ Don't confuse ML (learns from data) with traditional programming (explicit rules)

### Quick Quiz
1. A bank wants to identify fraudulent transactions → **Anomaly Detection**
2. What's the relationship between AI, ML, and DL? → **DL ⊂ ML ⊂ AI (nested subsets)**
3. Extracting text from scanned documents uses which workload? → **Computer Vision (OCR)**
4. ChatGPT-style applications fall under which workload? → **Generative AI / NLP**
